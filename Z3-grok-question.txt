# Вопрос для Grok: Создание Apple ID для development signing из России

## Контекст
Нужно создать **free Apple ID** для iOS development signing (не paid Developer Program).
Разработчик находится в **России**, есть VPN.

## Цель
Использовать free Apple ID для:
1. Создания development certificate в Xcode
2. Создания provisioning profile на developer.apple.com
3. Signing IPA в GitHub Actions для тестирования на iPhone

## Наша ситуация
- Разработчик: Россия, есть VPN (разные страны)
- Mac: удалённый доступ, находится в другой стране
- iPhone: российский, iOS 18.4.1
- Цель: протестировать Yandex Mobile Ads SDK
- Бюджет: $0 (только free tools)

## Важно
Нужна **проверенная инструкция на ноябрь 2024**, которая точно работает из России.
